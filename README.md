# Flask Book Tracker

A web application for tracking reading progress and managing personal library.

## Current Features
- Flask web server with multiple routes
- JSON API endpoints
- Responsive design (tested on desktop and mobile)

## Tech Stack
- Python 3.x
- Flask
- SQLite (planned)
- HTML/CSS (planned)

## Setup
```bash
pip install flask
python app.py
```
